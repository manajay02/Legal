## ğŸ’¡ Tips & Tricks

### Quick Customization
- **Change colors**: Edit CSS variables in style.css (lines 7-18)
- **Change text**: Edit HTML templates directly
- **Add features**: Follow existing component patterns

### Testing
- **Local**: Run `python app.py` and visit `http://localhost:5000`
- **Responsive**: Use browser DevTools (F12) to resize
- **Mobile**: Use responsive design mode or test on real device
- **Accessibility**: Use keyboard navigation, test with screen reader

### Troubleshooting
- **CSS not loading**: Clear browser cache (Ctrl+Shift+Delete)
- **Port in use**: Change port in app.py last line
- **Layout issues**: Check viewport meta tag in HTML
